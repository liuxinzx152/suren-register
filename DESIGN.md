# DESIGN.md - 素人登记小程序

## 项目与用户画像
- 推广部内部工具，高频使用的数据管理应用
- 用户场景：外勤买手开发人员在手机上快速登记、查找、管理素人信息
- 核心诉求：替代Excel，快速录入和检索，移动端随时可用

## 气质与意象
- 关键词：高效、克制、信赖感
- 意象：一张整洁的金属工作台，工具各归其位，拿取顺手。没有多余装饰，每个元素都有明确功能
- 风格定位：现代工具类 App（类似飞书/Notion Mobile），干净利落，信息密度适中

## 配色方案
- 主色：深青色 (Teal) `oklch(0.55 0.15 180)` — 沉稳专业，区别于通用科技蓝
- 主色-亮：用于按钮、链接、活跃状态
- 背景：极浅灰白 `oklch(0.98 0 0)` — 减少手机屏幕刺眼感
- 卡片：纯白 `#ffffff` — 信息区块清晰分离
- 文字主色：深灰 `oklch(0.2 0 0)` — 非纯黑，减少对比疲劳
- 文字辅色：中灰 `oklch(0.55 0 0)` — 次要信息
- 边框：浅灰 `oklch(0.92 0 0)` — 轻分隔不抢眼
- 状态色：
  - 待处理：琥珀色 `oklch(0.75 0.15 80)`
  - 进行中：天蓝色 `oklch(0.65 0.15 220)`
  - 已完成：翠绿色 `oklch(0.65 0.15 155)`
  - 已取消：浅红色 `oklch(0.65 0.12 20)`
- 危险操作：红色 `oklch(0.577 0.245 27.325)`

## 字体排版
- 字体族：系统默认（PingFang SC / Microsoft YaHei / system-ui）
- 标题：16-18px, font-weight 600
- 正文/表单：14-15px, font-weight 400
- 辅助文字：12-13px, font-weight 400, 使用 muted-foreground 色
- 行高：1.5-1.6 保证可读性

## 布局与响应式
- 桌面端优先，最大宽度 1600px 居中
- 顶部固定导航栏（Logo + 导航链接 + 用户信息）
- 数据表格展示，一行显示多个字段
- 搜索筛选区域横向排列在表格上方
- 表单使用右侧抽屉（Sheet）形式
- 详情使用弹窗（Dialog）展示
- 适配常见PC分辨率（1920x1080、1366x768等）
- 手机端保持基本可用性（响应式降级）

## 交互与状态
- 按钮点击有明确反馈（active scale 或背景变化）
- 列表项点击进入详情，支持左滑删除（桌面端显示删除按钮）
- 表单提交后自动返回列表并显示成功提示
- 加载状态使用骨架屏（Skeleton）
- 空状态使用图标 + 文字引导

## 设计禁忌
- 不要使用渐变背景或花哨装饰
- 不要使用大面积纯蓝色（避免与通用后台系统雷同）
- 不要在列表卡片中塞入过多信息，保持一屏可读
- 不要使用小于 12px 的字体
- 不要使用动画过渡超过 300ms
